# ⚽ Football Pro

A free real-time multiplayer 3D football game built with Godot — download for Windows, Linux, or macOS and play with friends online!

🇻🇳 [Đọc bằng tiếng Việt →](README.vi.md)

---

## 🎮 Download & Play

Go to the **[Releases page](https://github.com/golviral/FootBallPro/releases/latest)** and download the file for your operating system:

| Platform | File to download |
|----------|-----------------|
| 🪟 Windows | `fbgx.exe` (or `fbgx_windows.zip`) |
| 🐧 Linux | `fbgx.x86_64` (or `fbgx_linux.zip`) |
| 🍎 macOS | `fbgx.zip` |

**How to install:**
1. Download the file for your OS above.
2. Unzip it if needed.
3. Run the game — no installation required!

> **Web version:** Football Pro also has a web build you can play directly in your browser without downloading anything — check the [latest release](https://github.com/golviral/FootBallPro/releases/latest) for the web build files.

---

## 🕹️ How to Play

1. **Launch** the game.
2. **Create an account** or log in with your existing credentials.
3. **Set up your club** — choose your team name, formation, and player appearance.
4. **Join or create a match lobby** — play against friends or AI opponents.
5. **Enjoy the match!** — score goals and climb the leaderboard.

## 🎯 Controls

### ⌨️ Keyboard

| Action | Key(s) |
|--------|--------|
| Move | Arrow keys |
| Sprint | **W** |
| Shoot | **D** |
| Pass / Switch player | **S** |
| Lob (with ball) / Slide tackle (no ball) | **A** |
| Through-pass | **E** |
| Double pressure (two players press at once) | **S + D** |
| Open tactics menu | **T** |

### 📱 Mobile / Touch

Use the on-screen joystick to move and the action buttons to shoot, pass, and more.
