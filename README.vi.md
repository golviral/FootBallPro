# ⚽ Football Pro

Trò chơi bóng đá 3D nhiều người chơi trực tuyến được xây dựng bằng Godot — hoàn toàn miễn phí! Tải về cho Windows, Linux hoặc macOS và cùng chơi với bạn bè.

🇬🇧 [Read in English →](README.md)

---

## 🎮 Tải Về & Chơi

Truy cập trang **[Releases](https://github.com/golviral/FootBallPro/releases/latest)** và tải file phù hợp với hệ điều hành của bạn:

| Hệ điều hành | File cần tải |
|--------------|-------------|
| 🪟 Windows | `fbgx.exe` (hoặc `fbgx_windows.zip`) |
| 🐧 Linux | `fbgx.x86_64` (hoặc `fbgx_linux.zip`) |
| 🍎 macOS | `fbgx.zip` |

**Cách cài đặt:**
1. Tải file phù hợp với hệ điều hành của bạn.
2. Giải nén nếu cần.
3. Chạy trò chơi — không cần cài đặt thêm bất cứ thứ gì!

> **Phiên bản Web:** Football Pro cũng có bản web để chơi trực tiếp trên trình duyệt mà không cần tải về — xem [bản phát hành mới nhất](https://github.com/golviral/FootBallPro/releases/latest) để tải file web.

## 🕹️ Hướng Dẫn Chơi

1. **Khởi động** trò chơi.
2. **Đăng ký tài khoản** hoặc đăng nhập nếu đã có tài khoản.
3. **Thiết lập câu lạc bộ** — chọn tên đội, đội hình và ngoại hình cầu thủ.
4. **Tham gia hoặc tạo phòng đấu** — chơi với bạn bè hoặc đối thủ AI.
5. **Bắt đầu trận đấu!** — ghi bàn và chinh phục bảng xếp hạng.

## 🎯 Điều Khiển

### ⌨️ Bàn phím

| Hành động | Phím |
|-----------|------|
| Di chuyển | Phím mũi tên |
| Chạy nhanh | **W** |
| Sút bóng | **D** |
| Chuyền / Đổi cầu thủ | **S** |
| Lob (có bóng) / Xoạc (không có bóng) | **A** |
| Chọc khe | **E** |
| Kèm 2 người cùng lúc | **S + D** |
| Mở menu chiến thuật | **T** |

### 📱 Điện thoại / Màn hình cảm ứng

Sử dụng joystick ảo để di chuyển và các nút hành động để sút bóng, chuyền bóng và nhiều hơn nữa.
